# 02 - Technical Stack

---

## 1. Programming Languages

- **Backend :** Aucun backend dédié (application purement front-end servie en statique).
- **Frontend :** HTML5, CSS3, JavaScript (ES6+).

## 2. Frameworks

- **Backend :** N/A (pas de framework côté serveur).
- **Frontend :** Aucun framework majeur (vanilla JavaScript, DOM API, CSS personnalisés).

## 3. Database

- **Type :** Aucune base de données serveur ; persistance locale via `localStorage` du navigateur.

## 4. Key Libraries

- **Frontend :**
  - APIs Web natives : DOM, `localStorage`, `FileReader`, `navigator.clipboard`.
  - Aucune dépendance externe obligatoire (pas de framework JS ni de gestionnaire d’état).
- **Backend :**
  - Non applicable (aucun code backend spécifique au projet).

## 5. Development & Deployment Environment

- **Environnement local :** WampServer sur Windows, projet servi depuis `d:/wamp64/www/suprompts`.
- **Serveur :** Serveur web statique (Apache via Wamp ou tout hébergeur de fichiers statiques).
- **Tests :** Principalement tests manuels via le navigateur (pas encore de framework de tests automatisés).
